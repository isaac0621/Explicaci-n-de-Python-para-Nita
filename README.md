# 🚀 Curso Interactivo de Python – Nita

Bienvenida Nita 👋  
Este no es un curso normal. Es un **entrenamiento práctico paso a paso**.

Regla del juego:
👉 Leés → ejecutás → respondés → avanzás

La idea es que puedas aprender la teoría con base a la práctica y así puedas ejecutarla de manera sencilla, esto para que puedas responder a problemas y situaciones en momentos importantes, esto para que el avance sea continuo y más preciso en el día a día.
