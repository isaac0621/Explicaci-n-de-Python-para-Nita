# 2. 📊 Tipos de datos

# 🛒 Carrito de compras

## 🎯 Objetivo
Aplicar `while` + acumulación de datos.

## 🧾 Enunciado
Crear un sistema que permita ingresar precios de productos hasta que el usuario decida terminar.

## 📌 Requisitos
- Pedir precios en bucle
- Sumar el total
- Preguntar si desea continuar
- Terminar cuando el usuario diga "no"

## ✅ Criterio de éxito
Se muestra el total correcto al finalizar.
