# 🔐 Proyecto 4 - Login simple

## 🎯 Objetivo
Trabajar validaciones con `if`.

## 🧾 Enunciado
Crear un sistema que pida una contraseña y valide si es correcta.

## 📌 Requisitos
- Pedir contraseña
- Comparar con valor fijo ("1234")
- Mostrar acceso permitido o denegado

## ✅ Criterio de éxito
El sistema responde correctamente según la contraseña.
