# ⌨️ Ejercicios Resueltos — `input()` en Python

---

## Ejercicio 1 — Pedir nombre

**Enunciado:** Pedile al usuario que ingrese su nombre y guardalo en una variable.

```python
nombre = input("Ingresá tu nombre: ")
```

**Ejemplo de ejecución:**
```
Ingresá tu nombre: Ana
```

## Ejercicio 5 — Mostrar edad ingresada

**Enunciado:** Pedile al usuario su edad y mostrala con un mensaje.

```python

```

**Ejemplo de ejecución:**
```
Ingresá tu edad: 25

```

## Ejercicio 6 — Pedir dos datos

**Enunciado:** Pedile al usuario su nombre y su ciudad en variables separadas.

```python
nombre = input("Ingresá tu nombre: ")
ciudad = input("Ingresá tu ciudad: ")
```

**Ejemplo de ejecución:**
```
Ingresá tu nombre: Ana
Ingresá tu ciudad: Buenos Aires
```

## Ejercicio 7 — Mostrar ambos datos

**Enunciado:** Pedí nombre y ciudad al usuario y mostralos juntos en un mensaje.

```python
nombre = input("Ingresá tu nombre: ")
ciudad = input("Ingresá tu ciudad: ")
print("Nombre:", nombre, "- Ciudad:", ciudad)
```

**Ejemplo de ejecución:**
```
Ingresá tu nombre: Ana
Ingresá tu ciudad: Buenos Aires
Nombre: Ana - Ciudad: Buenos Aires
```

## Ejercicio 9 — Frase con input

**Enunciado:** Pedile al usuario su nombre y mostrá un mensaje personalizado de bienvenida.

```python
nombre = input("Ingresá tu nombre: ")
print("Hola,", nombre, ", bienvenido a Python")
```

**Ejemplo de ejecución:**
```
Ingresá tu nombre: Ana
Hola, Ana , bienvenido a Python
```
