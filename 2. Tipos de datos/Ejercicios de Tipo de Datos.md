# 🗂️ Ejercicios Resueltos — Tipos de Datos en Python

---

## Ejercicio 1 — Crear string

**Enunciado:** Creá una variable de tipo `str` y asignale cualquier palabra o frase.

```python
saludo = "Hola, mundo"
```

---

## Ejercicio 2 — Crear int

**Enunciado:** Creá una variable de tipo `int` y asignale cualquier número entero.

```python
cantidad = 42
```

---

## Ejercicio 3 — Crear float

**Enunciado:** Creá una variable de tipo `float` y asignale cualquier número decimal.

```python
precio = 9.99
```

---

## Ejercicio 4 — Mostrar string

**Enunciado:** Creá una variable de tipo `str` y mostrala en pantalla.

```python
saludo = "Hola, mundo"
print(saludo)
```

**Salida:**
```
Hola, mundo
```

## Ejercicio 7 — Tres tipos distintos

**Enunciado:** Creá tres variables, una de cada tipo: `str`, `int` y `float`.

```python
nombre = "Ana"
edad = 25
altura = 1.65
```

---

## Ejercicio 8 — Mostrar todos

**Enunciado:** Mostrá las tres variables del ejercicio anterior en pantalla.

```python
nombre = "Ana"
edad = 25
altura = 1.65
print(nombre)
print(edad)
print(altura)
```

**Salida:**
```
Ana
25
1.65
```

**Enunciado:** Creá una variable llamada `temperatura` con un valor decimal y mostrala.
